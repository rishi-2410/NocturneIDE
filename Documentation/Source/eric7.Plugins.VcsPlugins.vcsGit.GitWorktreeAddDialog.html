GitWorktreeAddDialog.getParameters

getParameters()

Public method to get the entered parameters for the 'git worktree add' operation.

Return:
dictionary containing the entered data. This dictionary has these keys.
  • path: path for the new worktree
  • branch: name for the worktree branch
  • force_branch: enforce creating the branch
  • lock: flag indicating to lock the worktree
  • lock_reason: optional reason string for the lock
  • detach: flag indicating to detach the HEAD in the new worktree
  • commit: commit to check out in the new worktree (branch, tag, commit ID or empty for HEAD)
  • force: flag indicating to enforce the worktree creation
Return Type:
dict
