eric7.MicroPython.Devices.CircuitPythonUpdater

Package implementing the updater and associated dialogs.

Modules

CircuitPythonUpdaterInterface Module implementing an interface to the 'circup' package.
CircupFunctions Module implementing variants of some 'circup' functions suitable for 'eric-ide' integration.
RequirementsDialog Module implementing a dialog to generate a requirements file.
ShowBundlesDialog Module implementing a dialog showing the available bundles and their modules.
ShowInstalledDialog Module implementing a dialog to show the modules installed on the connected device.
ShowOutdatedDialog Module implementing a dialog to show outdated modules of a connected device.
