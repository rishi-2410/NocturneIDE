eric7.CycloneDXInterface

Package implementing the interface to the CycloneDX tool for the generation of Software Bill of Materials files.

Modules

CycloneDXConfigDialog Module implementing a dialog to configure the CycloneDX SBOM generation.
CycloneDXMetaDataDialog Module implementing a dialog to edit the metadata of the CycloneDX SBOM.
CycloneDXUtilities Module implementing the interface to CycloneDX.
