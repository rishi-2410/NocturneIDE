eric7.Snapshot

Package implementing the modules for the screenshot utility.

Modules

SnapWidget Module implementing the snapshot widget.
SnapshotDefaultGrabber Module implementing a grabber object for non-Wayland desktops.
SnapshotFreehandGrabber Module implementing a grabber widget for a freehand snapshot region.
SnapshotModes Module implementing the snapshot mode enumeration.
SnapshotPreview Module implementing the snapshot preview label.
SnapshotRegionGrabber Module implementing a grabber widget for a rectangular snapshot region.
SnapshotTimer Module implementing the snapshot timer widget.
SnapshotWaylandGrabber Module implementing a grabber object for non-Wayland desktops.
