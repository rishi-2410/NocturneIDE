eric7.MicroPython.WifiDialogs

Package implementing WiFi related dialogs.

Modules

WifiApConfigDialog Module implementing a dialog to configure the Access Point interface.
WifiApStationsDialog Module implementing a dialog showing the currently connected stations (clients).
WifiConnectionDialog Module implementing a dialog to enter the parameters needed to connect to a WiFi network.
WifiController Module implementing the WiFi related functionality.
WifiCountryDialog Module implementing a dialog to enter the country code for the WiFi interface.
WifiNetworksWindow Module implementing a dialog showing the available WiFi networks.
WifiStatusDialog Module implementing a dialog to show the WiFi status of the connected device.
